      <p>A food map embedded in Douyin that connects short-form content with local-life discovery, helping users turn inspiration into nearby dining decisions.</p>
      <ul>

      </ul>
    </article>
    <article class="project-card">

        <span class="pub-tag pub-tag--soft">Workflow generation</span>
      </div>
      <h3>Agentify</h3>
      <p>A natural-language-driven framework for generating LLM workflows. Built on Dify and LangGraph, it uses a dual-agent architecture to plan, serialize, and render workflows into deployable HTML interfaces.</p>
      <ul>
        <li>Reached 100+ GitHub stars.</li>
        <li>Focused on zero-code agent and workflow composition.</li>
      </ul>
    </article>
